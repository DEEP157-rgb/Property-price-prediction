# Property Price Prediction

## Overview
This project predicts real estate property prices using machine learning techniques.

## Workflow
1. Data Cleaning & Preprocessing
2. Exploratory Data Analysis (EDA)
3. Feature Engineering
4. Model Building (Linear Regression / Random Forest)
5. Model Evaluation

## Features Used
- Area / Sqft
- Bedrooms
- Bathrooms
- Location
- Property Age

## Model
- Regression model used for price prediction
- Evaluated using R² and RMSE

## Use Case
- Helps buyers estimate fair price
- Helps sellers set optimal listing price
- Supports real estate investment decisions

## Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib / Seaborn
